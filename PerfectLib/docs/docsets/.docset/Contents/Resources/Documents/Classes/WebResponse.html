WebResponse

public class WebResponse

Represents an outgoing web response. Handles the following tasks: - Management of sessions - Collecting HTTP response headers & cookies. - Locating the response template file, parsing it, evaluating it and returning the resulting data. - Provides access to the WebRequest object.
